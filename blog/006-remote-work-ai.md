## The New Remote Work Stack

### Communication (Async-First)
**Tools:** Loom + Notion + Slack

- **Loom** for video updates (no more Zoom fatigue)
- **Notion** for documentation (single source of truth)
- **Slack** for urgent only (protect deep work)

**Result:** 70% fewer meetings, same output.

### Deep Work (AI-Enhanced)
**Tools:** Claude + Cursor + Brain.fm

- **Claude** for research and writing
- **Cursor** for coding
- **Brain.fm** for focus music

**Result:** 4-hour deep work blocks, daily.

### Collaboration (Timezone-Agnostic)
**Tools:** Figma + GitHub + Linear

- **Figma** for design (real-time collaboration)
- **GitHub** for code (async reviews)
- **Linear** for project management

**Result:** Work across timezones seamlessly.
